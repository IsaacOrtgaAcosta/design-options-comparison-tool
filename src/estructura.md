##Funcionalidades
1. Formulario creación diseño
    1.1 Manejo de errores formulario (inputs vacíos, tipo de datos, etc.)
    1.2 Envío de datos a API
    1.3 Manejo error/éxito
2. Importar datos
    2.1 Fetch
    2.2 Ordenar datos, manejo de excepciones (mapeo)
    2.3 Guardar en BD
3. Construcción de cards con datos de diseño
    3.1 Constuir diseño card
    3.2 Manejo de muestreo (mapear)
    3.3 Manejo de excepciones
4. Construcción modal para mostrar detalles
    4.1 Fetch a API para traer todos los datos
    4.2 Mapear datos
    4.3 Manejo de excepciones (reutilizable)
5. Construcción portal para comparación
    4.1 Fetch a API para traer los datos por id de cada uno que ha seleccionado
    4.2 Mapeo de datos
    4.3 Manejo de excepciones (reutilizable)
